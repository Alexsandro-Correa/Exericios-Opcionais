# Exercicio Opcional 04

## Uma parede em formato retangular, cuja altura e a largura precisa ser coberta por azulejos também retangulares. O azulejo retangular tem as dimensões de  altura e largura. 

##Escreva um programa que:
> -  leia as quatro medidas;
> -  calcule e imprima quantos azulejos com as medidas dadas são necessários para cobrir a parede dada.

----------------------------------------------------

## Programa simples criado usando apenas:
> - Variáveis
> - Scanner
> - Syso
